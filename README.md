## 클래스!

덕 타이핑 (Duck Typing)
: 인터페이스에 대입되는 객체가 해당 인터페이스 상속-구현체가 아니어도 메서드만 같다면 상속-구현된 것으로 인정해주는 문법

선택적 프로퍼티 (Optional Property)
: 인터페이스의 프로터피는 반드시 구현되어야 하지만 선택적으로 필요한 경우가 있다. 선택적 프로퍼티는 프로퍼티 명 뒤에 ? 을 붙이고, 이때는 생략하여도 에러가 발생하지 않는다.
